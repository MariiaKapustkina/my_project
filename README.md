# 🌞 Horoscope App

A simple React application for horoscopes, built with **React + Vite + Tailwind CSS**.  
The app includes multiple pages, dark/light theme support, and a custom animated loader.  

---

## 🚀 Tech Stack
- ⚛️ React
- 🎨 Tailwind CSS
- 🛠️ Vite
- 🌙 Context API (theme switching)
- 🔮 React Router

---
